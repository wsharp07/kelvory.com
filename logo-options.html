Identity Explorations

Rounded K and abstract symbol studies for Kelvory.

This round follows your references more directly: one monoline `K` inspired by the rounded `M`, plus three more abstract marks pulled from the halo, eclipse, and triad symbol language in the inspiration set.

Option 1: Pulse K

Alt K
Rounded monoline K

A detached node and thick rounded strokes echo the `M` reference without turning it into a copy. It stays clean but still has personality.

Monoline / rounded / closest to your ask

Why I think you’ll like it: it gives you the clearer `K` direction you asked for, but with a softer, more contemporary silhouette than the angular options we tried before.

If the brand needs a letterform, this is the strongest starting point in this set.

Option 2: Sensor Halo

Strongest Abstract
Scanner / core / system

This takes the circular technical reference and strips it down into something cleaner and more logo-like, while still feeling like a sensing or control core.

Technical / iconic / futuristic

Why I think you’ll like it: it feels advanced and ownable without relying on a letter, which may give Kelvory more room to become its own thing.

This is my favorite abstract option in the new set.

Option 3: Eclipse Ring

Selected
Halo / aperture / signal

The simplest symbol here: a precise ring with a slight flare. It feels more premium and cinematic than overtly technical.

Minimal / premium / atmospheric

Why I think you’ll like it: it is the cleanest and most elegant abstract mark of the group, and it pairs naturally with the site’s darker, more atmospheric brand mood.

Choose this one if you want a quieter symbol that still feels futuristic.

Option 4: Triad Node

Most Emblematic
Triad / node / command

Inspired by the triangular symbol reference, this one feels more like an advanced insignia or systems emblem than a conventional app icon.

Symbolic / bold / insignia-like

Why I think you’ll like it: it has the strongest “skunkworks lab” energy in the group and could work well if you want the identity to feel a little more clandestine and elite.

Choose this one if you want the brand to feel more like a mark or badge than a software logo.

My read after vetting them

  • `Pulse K` is the best direct answer to your request for a mark like the rounded `M`, but as a `K`.
  • `Eclipse Ring` is the selected production direction because it is the cleanest premium abstraction in the set.
  • `Sensor Halo` is the strongest abstract option if you want something more overtly technical.
  • `Triad Node` is the boldest emblem if you want the identity to feel more covert and symbolic.

These are still exploration assets only, so we can choose the best direction before replacing the production logo files.
